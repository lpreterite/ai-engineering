# 版本比对

> SKILL.md 步骤 1 时加载。版本比对规则和差异判断。

## 版本格式

所有版本号采用语义化格式：`v{major}.{minor}`

- v0.1 - v0.9：草稿阶段
- v1.0 - v1.9：正式发布后的小版本
- v2.0+：重大修订

## 比对规则

| 条件 | 判定 |
|------|------|
| 当前版本 < 上游版本 | `needs-update` |
| 当前版本 == 上游版本 | `up-to-date` |
| 当前版本 > 上游版本 | 异常，标记检查 |

## 差异状态

| 状态 | 含义 | 策略 |
|------|------|------|
| `up-to-date` | 已是最新 | 跳过 |
| `needs-update` | 上游有新版本 | 按定制情况选策略 |
| `conflict` | 两者都有变更 | 三路合并 |
| `unknown` | 版本号缺失或格式异常 | 人工检查 |
