## 安全红线

- 禁止硬编码 API Key、数据库密码等敏感信息
- 禁止在日志中输出用户敏感信息（密码、Token）
- 所有用户输入必须校验和转义（防 XSS/SQL 注入）
- 敏感操作必须记录审计日志
