# 设计规范完整示例

> SKILL.md 步骤 2 时加载。设计规范的完整模板示例。

## 颜色系统模板

| 角色 | 色值 | 用途 |
|------|------|------|
| Primary | #4A90D9 | 主按钮、链接、品牌色 |
| Background | #FFFFFF | 页面背景 |
| Surface | #F5F5F5 | 卡片、容器背景 |
| Text Primary | #333333 | 正文 |
| Text Secondary | #666666 | 辅助文字 |
| Error | #E74C3C | 错误提示 |
| Success | #2ECC71 | 成功状态 |

## 断点定义

| 断点 | 宽度 | 目标设备 |
|------|------|----------|
| xs | < 576px | 手机 |
| sm | 576px - 768px | 大屏手机/小平板 |
| md | 768px - 1024px | 平板 |
| lg | 1024px - 1440px | 桌面 |
| xl | > 1440px | 大屏桌面 |

## 设计稿检查清单

```
□ 覆盖所有页面
□ 覆盖所有状态（默认/悬停/点击/禁用）
□ 覆盖空态、加载态、错误态
□ 标注规范完整
□ 间距、颜色、字体有明确说明
□ 尺寸规格覆盖响应式断点
□ 交互说明清晰
□ 图标资源齐全
□ 符合设计系统规范
□ 可访问性考虑
```
