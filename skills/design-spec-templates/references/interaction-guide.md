# 交互说明指南

> SKILL.md 步骤 3 时加载。交互说明编写规范。

## 交互说明模板

### 组件状态覆盖

| 组件 | 状态 | 说明 |
|------|------|------|
| 按钮 | 默认/悬停/点击/禁用/加载中 | 每种状态的颜色、光标、反馈 |
| 输入框 | 默认/聚焦/输入中/错误/禁用 | 边框颜色、提示文字、错误信息 |
| 下拉菜单 | 收起/展开/选中/空结果 | 展开方向、选中高亮、滚动 |
| 弹窗 | 显示/关闭/加载 | 出现动画、关闭方式、遮罩层行为 |

### 过渡动画说明

```
- 页面切换：右滑进入（300ms ease-out）
- 弹窗出现：淡入 + 上移（200ms ease-in-out）
- 列表加载：逐条淡入（每项间隔 50ms）
- 按钮点击：缩放 0.95（100ms）
```

## 响应式适配说明

| 组件 | xs | sm | md | lg+ |
|------|----|----|----|-----|
| 导航栏 | 汉堡菜单 | 汉堡菜单 | 全展开 | 全展开 |
| 侧边栏 | 隐藏 | 隐藏 | 可折叠 | 固定显示 |
| 卡片列表 | 1 列 | 2 列 | 3 列 | 4 列 |

## 设计标注要求

- 使用 Zeplin/Figma 导出标准标注
- 标注包含：间距、字号、字重、行高、颜色色值、圆角
- 特殊效果（阴影、渐变、模糊）需额外说明
